Add SQS Dead Letter Queue

Capture all failed Lambda events. No data silently lost in a production pipeline at this scale.

Impact: Critical reliability · ~$5/mo cost

Redshift Serverless

Auto-scales compute to match query demand. Eliminates idle provisioned cluster cost during off-peak hours.

💰 Save ~$400/mo
